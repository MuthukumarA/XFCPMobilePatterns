Before you begin

This class is a cross-platform class for Windows, iOS and Android. To complete the coding exercises, you will need a Xamarin development environment with either a simulator/emulator or a device to run the exercise on. Please read the setup requirements if you need help installing the Xamarin development environment.

Overview

This session builds on the introduction class and goes deeper into architectural patterns used to abstract our dependencies which can help with cross platform development as well as unit testing your shared code.

Objectives

  1. Working with Dependencies in a Loosely-Coupled fashion

Lecture

XAM300 - Advanced Cross Platform Mobile Development.

Exercises

  1. Using the Factory Pattern
    Exercise
    Create a dependency through the Factory pattern.
  2. Build a Service Locator
    Exercise
    Add an abstraction and Service Locator into our project to find and use a platform-specific feature.
  3. Using Dependency Injection
    Exercise
    Switch to a DI/IoC container and inject our dependency into our code.
